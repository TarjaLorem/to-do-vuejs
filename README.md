# TodoApp

This project was generated with vue-cli version 3.0.5.

## Project setup
```
npm install
```

### Development server
```
npm run serve
```
The app will automatically reload if you change any of the source files.

### Compiles and minifies for production
```
npm run build
```

### Run your tests
```
npm run test
```

### Lints and fixes files
```
npm run lint
```
